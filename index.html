Metadatamatic

Metadatamatic is a simple Web application that allows you to create the description of a dataset. It is being provided as a quick and easy way for you to create your own dataset description. Simply work through the forms on this page and complete whichever details you'd like to add to your description. If you enter the URL of an endpoint, you will be able to extract some of the details of the dataset automatically. The final dataset description is displayed at the bottom of the page. It can be downloaded as a file or copied and pasted into an RDF file.

By using this service, you accept that the data you input into Metadatamatic may be used by the WIMMICS team, exclusively for research purposes. Do not input sensitive private information in the fields. If you wish for your information to be removed from the data retrieved by the WIMMICS team, please contact us with the details of your information.
